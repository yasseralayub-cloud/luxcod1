# LuxCod — Premium Digital Agency Website

A high-end, conversion-focused single-page website for **LuxCod** digital agency.

## Features

- Ultra-modern dark luxury UI with gold & purple accents
- Fully responsive mobile-first design
- Arabic/English language toggle with full RTL/LTR support
- Animated hero section with particle canvas background
- Dynamic portfolio — edit `data.js` to add/remove projects
- Testimonials slider — edit `data.js` to update reviews
- Contact form via FormSubmit (sends to luxcode3@gmail.com)
- Floating WhatsApp button with pre-filled message
- Smooth scroll, micro-interactions, hover effects
- Zero dependencies — pure HTML, CSS, JavaScript

## File Structure

```
luxcod1/
├── index.html      # Main HTML structure
├── style.css       # All styles (dark luxury theme)
├── app.js          # JavaScript (interactions, slider, form)
├── data.js         # EDIT THIS to update portfolio & testimonials
└── README.md
```

## Editing Content

Edit `data.js` to update portfolio projects and testimonials without touching HTML.
